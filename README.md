# 🛍️ ShopEase - E-Commerce Mobile App

A complete cross-platform e-commerce mobile application built with React Native (Expo) and Node.js backend.

## 📱 Demo Features

- View products without logging in
- User registration and login with JWT
- Browse products with categories
- Add products to shopping cart
- Update cart quantities
- Place orders with shipping details
- View order history in profile

## 🏗️ Tech Stack

### Frontend (React Native Expo)
- **Framework:** React Native with Expo
- **Navigation:** React Navigation (Stack + Bottom Tabs)
- **HTTP Client:** Axios
- **Storage:** Expo SecureStore + Memory Storage
- **Icons:** React Native Vector Icons

### Backend (Node.js)
- **Runtime:** Node.js with Express
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT (JSON Web Tokens)
- **Password Hashing:** Bcryptjs
